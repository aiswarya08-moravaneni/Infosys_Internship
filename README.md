# 📊 HotelRevAI – AI Driven Revenue Analysis for Hotels

## 🚀 Project Overview
HotelRevAI is a data analytics project developed during the Infosys Springboard Virtual Internship 6.0.  
The project focuses on analyzing hotel booking data to uncover revenue trends, customer behavior, and booking patterns using Power BI.

It helps hotel management make data-driven decisions to improve business performance.

---

## 🎯 Objectives
- Analyze hotel booking data for insights
- Monitor revenue performance 📈
- Study occupancy trends 🏨
- Understand customer segments 👥
- Analyze booking cancellations ❌
- Predict future revenue using forecasting 🔮

---

## 🛠️ Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Kaggle Dataset
- Data Cleaning & Visualization Techniques

---

## 📂 Dataset
- Source: Kaggle
- Data Range: 2015 – 2017
- Includes:
  - City Hotel Data
  - Resort Hotel Data
  - Booking, revenue, and customer details

## 🔄 Project Workflow

### 1. Data Collection
- Collected dataset from Kaggle
- Explored structure and features

### 2. Data Cleaning & Preprocessing
- Handled missing values
- Removed duplicates
- Converted data types
- Created new columns (total nights, revenue)

### 3. Data Modeling
- Built relationships between tables
- Created measures using DAX
- Structured optimized data model

### 4. Dashboard Development
- Revenue Performance Dashboard  
- Occupancy & Demand Analysis  
- Cancellation Analysis  
- Revenue Forecasting Dashboard  

### 5. Visualization & Insights
- Designed interactive visuals
- Extracted meaningful business insights

### 6. Forecasting
- Used Power BI forecasting tools
- Predicted future revenue trends
